# Evara Advisory Website

## Local Development
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Vercel auto-detects Vite — just click Deploy
4. Add your custom domain in Settings → Domains → add `evaraadvisory.com.au`
5. Update DNS at VentraIP as instructed by Vercel
